MPMC Sequence Buffer Stepper

Visualize how the Vyukov/Dmitry algorithm coordinates multiple producers and consumers using ticket-based slot ownership with a sequence buffer.

Slot 0
empty (stamp=0)
Empty (stamp = i)
Filled (stamp = i + 1)
Consumed (stamp = i + capacity)
Initial state: All slots empty, stamps = 0. Ready for producer to claim ticket 0.

Note: Showing capacity=4, focusing on slot 0 which cycles through states as producers and consumers take turns.
